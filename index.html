Technology and art built with Script and Kabuki Toolkit.

Innovative Embedded-C++11 core with Standard Seam Tree Test Framework for firmware and software based on mbed and the Serial Chinese Room, Interprocess, and Telemetry (SCRIPT) Specification.

View Github

SCRIPT Specification

Serial Chinese Room, Interprocess, and Telemetry (SCRIPT) Specification defines the SCRIPT SDN Protocol, Crabs, and ASCII Data Types.

Kabuki Toolkit

A modern cross-platform C/C++11 toolkit for making intelligent connected technologies, plugins, games, servers, firmware, and art.

F2 Foundational Framework

A modern C++ nanoframework and Standard Seam Tree Unit Testing Framework for quick little diddles and big daddies.

Contribute to the Kabuki Starship!

We are always in need of new contributors. If you would like to sign up as a contributor, just make a pull request for an issue in the Kabuki Toolkit git repository.

Single Cross-platform Library

With Google ANGLE, Cairo, PortAudio, and Visual Studio Community Edition you can write one GUI and audio library for apps, games and plugins that runs on Windows, OSX, Linux, iOS, and Android!

Completely Free Software

All of the Kabuki Starship APIs are licensed under the Apache 2.0, BSD, MIT, and similar open-source software licenses, which is free to use for closed-source commercial purposes.

Virtual Control Technology

The Kabuki Starship is built completely around the Serial Chinese Room, Interprocess, and Telemetry (SCRIPT) Specification. Script allows for secure sophisticated mapping of controls and interprocess communication for the Internet of Things.

Make Robots!

With the Kabuki Starship, you can make all kinds of robots, and control them with IoT apps!
